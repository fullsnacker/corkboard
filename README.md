## 	:toolbox: Getting Started

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/fullsnacker/corkboard.git
```

Go to the project directory

```bash
  cd corkboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
